# Presentation

Ce folder et les sous répertoires, contiennent toutes les sources de la
présentation théorique. Comme je travaille sous Linux avec open office,
j'utilise peut être des polices de caractère qui ne sont pas disponibles chez
vous. Si nécessaire, elles sont disponibles pour installation dans ce repo.
Aussi, il peut être utile d'installer la palette de couleur que j'ai utilisé
pour que les modifications que vous apporteriez soient consistantes avec le
reste de la présentation.

## Etape 1: Installer les fonts

Elles sont toutes dans le folder fonts. 

## Etape 2: Installer la palette

Il suffit pour ca de copier le fichier PaletteXav.soc dans
`~/.config/libreoffice/4/user/config/`
